# Centro de Conciliación y Arbitraje Constructores de Paz

Sitio web corporativo profesional para centro de conciliación y arbitraje.

## Estructura del Proyecto

```
constructores-de-paz/
├── index.html
├── css/
│   └── styles.css
├── js/
│   └── script.js
├── images/
│   └── logo.png (colocar aquí el logo 1536x1024)
└── README.md
```

## Instalación

1. Colocar el logo en la carpeta `images/` con el nombre `logo.png`
2. Abrir `index.html` en un navegador web

## Características Implementadas

### Mejoras Visuales
- Logo integrado junto al título principal
- Footer completo con información de contacto, redes sociales y enlaces legales
- Botón de scroll to top funcional
- Animaciones suaves al hacer scroll
- Diseño responsive optimizado

### Estructura
- 3 miembros del equipo visibles (3 comentados en HTML)
- Secciones: Inicio, Historia, Organización, Equipo, Contacto
- Formulario de contacto con validación

### Funcionalidades
- Menú hamburguesa responsive
- Scroll suave entre secciones
- Validación de formulario en tiempo real
- Botones flotantes (WhatsApp y scroll top)
- Header con sombra dinámica al hacer scroll

## Navegadores Soportados

- Chrome (últimas 2 versiones)
- Firefox (últimas 2 versiones)
- Safari (últimas 2 versiones)
- Edge (últimas 2 versiones)

## Notas sobre el Logo

El logo debe ser PNG de 1536x1024 px y se colocará en `images/logo.png`.
El código está optimizado para mostrar el logo a 50px de altura manteniendo su proporción.

## Personalización

Para modificar colores, editar las variables CSS en `css/styles.css`:
- `--color-azul-oscuro`
- `--color-azul`
- `--color-amarillo`
